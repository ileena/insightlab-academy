# Content Rules

These rules define how AI should generate the academy content.

## Source Priority

Content must be derived from the following sources:

1. Uploaded PDF course materials
2. Udemy SQL Bootcamp course
3. Product Analytics Academy YouTube channel
4. Trusted documentation or analytics resources

Examples of trusted sources:

- Mixpanel documentation
- Amplitude documentation
- Google Analytics documentation
- Product analytics articles

---

## Strict Rules

1. Do not invent concepts that are not supported by references.
2. Every lesson must list the references used.
3. If a concept is unclear or missing from the sources, write:

   "Source material not provided yet."

4. Do not present generated assumptions as facts.
5. Keep explanations simple and educational.

---

## Lesson Structure

Each lesson should follow this structure:

1 Title

2 Why this concept matters

3 Explanation

4 Example

5 Practical takeaway



---

## Writing Style

The tone should be:

- clear
- structured
- beginner-friendly
- professional

Avoid:

- unnecessary complexity
- vague explanations
- unsupported claims

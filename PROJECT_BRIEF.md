# InsightLab – Data & Product Analytics Academy

## Project Overview

InsightLab is a learning platform designed to teach the fundamentals of:

- SQL
- Metrics & Dashboard Thinking
- Product Analytics

The goal of the academy is to create a structured learning resource for product analysts, product managers, and data analysts who want to develop strong analytical thinking for product development.

The platform will be built as an AI-assisted educational website and shared publicly through GitHub and LinkedIn as a portfolio project.

---

## Objectives

The academy aims to:

1. Provide structured educational content for learning analytics concepts.
2. Combine multiple educational sources into one structured learning path.
3. Demonstrate how AI-assisted development can help build educational platforms.
4. Showcase product thinking and knowledge architecture.

---

## Target Audience

The academy is designed for:

- aspiring product analysts
- product managers
- data analysts
- early career professionals learning analytics

---

## Learning Tracks

The academy will contain three main tracks.

### 1 SQL 
Teaching SQL concepts required for data analysis.

### 2 Metrics & Dashboard Thinking
Understanding how businesses measure performance.

### 3 Product Analytics
Understanding user behavior and product performance.
